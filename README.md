Overview

This project implements image segmentation using the K-Means clustering algorithm built completely from scratch in Python.
Instead of relying on machine learning frameworks such as scikit-learn or TensorFlow, the clustering logic is implemented manually using NumPy for numerical computation.

Image segmentation is an important technique in computer vision and digital image processing, where an image is divided into meaningful regions to simplify analysis.

The project demonstrates how an image can be segmented into groups of similar colors using the K-Means algorithm, which partitions data points into clusters based on distance from cluster centroids.
Project Features

K-Means clustering implemented from scratch

No machine learning libraries used

Image processing using NumPy and PIL

Visualization using Matplotlib

Elbow Method to determine optimal number of clusters

Color-based image segmentation

Visualization of individual color clusters
